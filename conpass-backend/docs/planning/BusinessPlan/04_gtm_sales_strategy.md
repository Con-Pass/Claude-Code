# 04. GTM・営業戦略

## 1. GTM戦略概要

### 1.1 市場参入の基本方針

ConPassは「紙も電子も契約書を一元管理できる唯一のクラウドサービス」として、BPOサービス統合という独自の差別化要素を武器に、中小企業の契約管理市場を段階的に攻略する。市場参入の基本方針は以下の3点に集約される。

1. **チャネルレバレッジの最大化**：自社の直販リソースだけに依存せず、士業・業界団体・ITディストリビューターを通じた間接チャネルを段階的に構築し、営業効率を飛躍的に高める
2. **BPO統合による参入障壁の構築**：スキャン・回収・アップロード・原本保管・機密処理といったBPOサービスとの統合は、純粋なSaaS競合が容易に模倣できない構造的優位性を形成する
3. **データネットワーク効果の発動**：契約データの蓄積量に比例してAIアシスタントの精度が向上し、将来的にはConPassスコアを金融・保険領域に展開することで、プラットフォームとしてのロックイン効果を実現する

5ヵ年の到達目標は、ARR 285,450千円（59期末）、顧客数254社、パートナー経由MRR比率20%以上とする。

### 1.2 5層利用者モデルに基づくフェーズ別戦略

ConPassの市場浸透は、5層の利用者モデルに基づくフェーズ別の段階的展開で推進する。各フェーズは独立ではなく、前フェーズの成果が次フェーズの推進力となる構造である。

| フェーズ | 時期 | 対象層 | 戦略の骨子 | 成功指標 |
|---------|------|--------|-----------|---------|
| Phase 1 | 55期〜56期 | 第2層：士業（税理士・弁護士） | 士業チャネルを確立し、顧問先企業への導入を加速。税理士の3点照合ニーズ、弁護士の法改正対応ニーズを起点とする | 士業パートナー50事務所、パートナー経由導入100社 |
| Phase 2 | 56期〜57期 | 第2層：監査法人 | 監査法人の監査証跡ニーズに応え、被監査企業への「導入強制力」を活用した面展開を実現する | Big4含む監査法人10社との提携、被監査企業200社導入 |
| Phase 3 | 57期〜58期 | 第3層：業界団体・協会 | OEM提供により数百〜数千社の一括導入を実現。不動産→介護→製造の順に展開 | OEM契約3業界団体、団体経由導入500社 |
| Phase 4 | 58期〜59期 | 第4層：金融機関・保険会社 | ConPassスコアを融資・保険評価に組み込み、導入インセンティブを創出する | 地銀3行との提携、ConPassスコア連携企業300社 |

なお、第1層（契約の当事者である中小企業）への直販は全フェーズを通じて並行して継続し、データの母数を拡大し続ける。第5層（行政・規制当局）は59期以降の中長期テーマとして、監査証跡の標準フォーマット化を働きかける。

### 1.3 BPO統合による差別化ポジショニング

ConPassの最大の差別化要素は、日本パープルが培ってきたBPOサービスとの統合にある。競合の電子契約サービスが「電子契約の締結」に特化する中、ConPassは紙の契約書も含めた全契約のライフサイクル管理を実現する。

**差別化の構造：**

```
┌─────────────────────────────────────────────────┐
│            ConPass クラウドプラットフォーム          │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐       │
│  │ 契約書管理 │  │ AIアシスタント│  │  分析・検索 │       │
│  └──────────┘  └──────────┘  └──────────┘       │
├─────────────────────────────────────────────────┤
│           BPO統合レイヤー（模倣困難）              │
│  スキャン ─ 回収 ─ アップロード ─ 原本保管 ─ 機密処理  │
├─────────────────────────────────────────────────┤
│          日本パープル 7,000社の既存基盤             │
│         20,000事業所との取引実績                   │
└─────────────────────────────────────────────────┘
```

このBPO統合は特に以下の場面で競争優位を発揮する。

- **導入時の移行コスト削減**：既存の紙契約書のスキャン・データ化をBPOとして丸投げ可能。他社では顧客自身が手作業で移行する必要がある
- **継続的な運用負荷の軽減**：紙で届く契約書の回収・スキャン・アップロードを自動化。完全電子化が難しい中小企業の現実に即した運用
- **セキュリティ担保**：原本保管と機密処理を一貫して提供。契約書のライフサイクル全体をカバー
- **士業チャネルとの親和性**：司法書士を中心に、原本保管＋スキャニング＋機密廃棄のパッケージ提案が可能

## 2. 直販チャネル戦略

### 2.1 展示会戦略

展示会はConPassにとって最大のリード獲得チャネルであり、55期ARR 4,176千円から59期ARR 91,368千円への成長を牽引する。

#### 年間計画

| 項目 | 55期 | 56期 | 57期 | 58期 | 59期 |
|------|------|------|------|------|------|
| 年間出展回数 | 5回 | 6回 | 6回 | 6回 | 6回 |
| 1回あたりリード獲得数 | 240件 | 280件 | 320件 | 340件 | 360件 |
| 商談化率 | 15% | 17% | 19% | 20% | 20% |
| 受注率 | 10% | 12% | 15% | 15% | 15% |
| 1回あたり受注数 | 3.6社 | 5.7社 | 9.1社 | 10.2社 | 10.8社 |
| 出展費用/回 | 180万円 | 190万円 | 200万円 | 200万円 | 200万円 |

#### ROI分析

- **現状**（55期）：リード獲得単価 7,083円、受注獲得単価 546,296円、受注率10%
- **目標**（57期以降）：TIP開発ブースト後に受注率15%へ向上、受注数5.4社/回以上を安定的に達成
- **投資回収**：1回出展180万円に対し、受注3.6社 x ARPA 50万円 = 年間売上1,800千円。ARPA 100万円達成時には3,600千円となり、投資回収期間は大幅に短縮

#### 改善施策

1. **TIP（Technology Integration Platform）開発ブーストとの連動**：製品デモの訴求力を強化し、商談化率・受注率を引き上げる。特にAIアシスタントのライブデモは来場者の関心を強く惹く
2. **展示会後のナーチャリング強化**：獲得リードに対するフォローアップシーケンスを体系化。展示会直後の3日以内コール→1週間以内ウェビナー招待→2週間以内トライアル提案の3段階フロー
3. **ブース設計の最適化**：BPOサービスとの統合を体感できるデモ環境を構築。「紙の契約書をその場でスキャン→即座にConPass上で検索可能」の実演
4. **出展展示会の選定精度向上**：過去の出展実績データに基づき、リード品質の高い展示会に集中投資。業界特化型展示会（不動産EXPO、医療福祉EXPO等）への出展も57期以降検討

### 2.2 リスティング広告

リスティング広告は安定的なリード獲得源であるが、現状の受注率4%が最大のボトルネックである。

| リード獲得単価 | 21,978円 | 18,000円 | 15,000円 |
| 受注獲得単価 | 879,121円 | 450,000円 | 300,000円 |
| 受注率 | 約4% | 8% | 10% |
| 月間リード獲得数 | - | 50件 | 80件 |

#### 受注率改善施策

1. **ランディングページの最適化**
   - 業種別LP（不動産、介護、製造等）の制作により、検索意図と着地先の一致度を向上
   - 導入効果の定量訴求を強化（年間360時間の業務削減、検索時間75%削減）
   - IT導入補助金対象であることの訴求を前面に出す

2. **キーワード戦略の精緻化**
   - 「契約書管理」等のビッグワードに加え、「紙 契約書 電子化」「契約書 スキャン サービス」等のロングテールキーワードを強化
   - 「電子契約」ではなく「契約管理」領域のキーワードに注力し、BPO統合の差別化を訴求
   - 士業向けキーワード（「税理士 契約書管理」「顧問先 契約書 一元管理」）の追加

3. **リードスコアリングの導入**
   - 獲得リードを業種・規模・行動データに基づきスコアリングし、営業リソースの配分を最適化
   - 高スコアリードへの即日架電体制を構築

4. **リターゲティングの強化**
   - サイト訪問者へのリターゲティング広告を実施
   - トライアル未完了者への再アプローチ広告を配信

## 3. 間接販売チャネル戦略

### 3.1 士業チャネル（Phase 1〜2）

士業チャネルはConPassのGTM戦略における最優先チャネルであり、1人の士業が複数の顧問先を横断的に管理するという構造を活かし、効率的な面展開を実現する。

#### 段階的攻略計画

**ステップ1：税理士チャネル（55期〜56期）**

税理士は約8万人の母数を持ち、ConPassとの親和性が最も高い。

- **訴求ポイント**：
  - 3点照合による請求と契約の整合性自動検証
  - AIアシスタントが契約本文と法令根拠付きで回答
  - 複数顧問先の契約を横断検索・一元管理
- **アプローチ方法**：
  - 税理士向けウェビナーの定期開催（月1回）
  - 税理士会支部への講演・セミナー提案
  - 税理士向け専門誌・メディアへの寄稿
  - 会計ソフトメーカー（freee、マネーフォワード等）との連携提案
- **目標**：56期末で税理士パートナー30事務所、顧問先企業100社導入

**ステップ2：弁護士チャネル（56期〜57期）**

- **訴求ポイント**：
  - 法改正時の影響契約自動リストアップ
  - 判例DBとの統合による契約リスク可視化
  - 顧問先企業の契約ポートフォリオ一元管理
- **アプローチ方法**：
  - 弁護士向けリーガルテックカンファレンスへの出展
  - 弁護士会への講演提案
  - リーガルテック分野のメディア連携
- **目標**：57期末で弁護士パートナー20事務所、顧問先企業50社導入

**ステップ3：監査法人チャネル（57期〜58期）**

監査法人は「導入強制力」を持つ唯一のチャネルであり、被監査企業への一括導入を実現できる。

- **訴求ポイント**：
  - 3点照合と監査証跡による全件検証自動化
  - 監査工数の大幅削減（推定40%以上）
  - 被監査企業の契約データへのセキュアなアクセス
- **アプローチ方法**：
  - Big4（EY、デロイト、KPMG、PwC）への個別提案
  - 中小監査法人への展示会・セミナーアプローチ
  - 監査法人向け専用プランの策定
- **目標**：58期末で監査法人10社提携、被監査企業200社導入

**ステップ4：その他士業への展開（57期以降）**

| 士業 | 訴求ポイント | 展開時期 |
|------|------------|---------|
| 弁理士 | ライセンス料突合、特許失効時の影響契約 | 57期〜 |
| 社会保険労務士 | プッシュ型法令変更監視 | 57期〜 |
| 行政書士 | 許認可と契約の紐付け自動化 | 58期〜 |
| 司法書士 | BPO統合（原本保管＋スキャニング＋機密廃棄） | 56期〜 |

#### 士業チャネルの収益モデル

```
1士業事務所 → 平均顧問先20社 → ConPass導入率30% → 6社/事務所
6社 × ARPA 60万円（士業経由はスタンダードプラン中心）= 3,600千円/事務所
```

50事務所 × 3,600千円 = 年間180,000千円のARRポテンシャル

## 4. 営業プロセスと組織

### 4.1 セールスファネル設計

ConPassの営業プロセスは、リード獲得から受注・オンボーディングまでの一貫したファネルとして設計する。

```
[認知]        [リード獲得]      [商談]        [受注]       [オンボーディング]
  │               │              │            │              │
  │  展示会        │  初回面談     │  提案・見積  │  導入支援     │
  │  リスティング   │  ニーズヒア   │  トライアル  │  BPO手配     │
  │  ウェビナー    │  リング      │  稟議支援   │  データ移行   │
  │  コンテンツ    │              │            │              │
  │  紹介         │              │            │              │
  ▼               ▼              ▼            ▼              ▼
 月間300件       月間50件        月間20件      月間3〜5件     累積管理
  └──────┬──────┘  └──────┬──────┘  └──────┬──────┘
     商談化率15%        受注率15%        オンボーディング
                                        完了率95%
```

#### ファネル各段階の施策

**リード → 商談（商談化率15%目標）：**
- 獲得から24時間以内の初回コンタクト（スピード対応の徹底）
- リードスコアリングによる優先順位付け
- 業種・規模に応じたナーチャリングコンテンツの自動配信

**商談 → 受注（受注率15%目標）：**
- 業種別の提案テンプレートの整備
- ROI試算ツールの提供（導入効果の定量化）
- IT導入補助金の申請支援による初期費用のハードル低減
- 無料トライアル（14日間）の積極活用
- 経営層向けの導入効果報告書テンプレート（稟議支援）

**受注 → オンボーディング（完了率95%目標）：**
- 導入キックオフミーティングの実施
- BPOサービス（スキャン・回収・アップロード）の手配
- 利用開始後30日間の集中サポート

## 7. リスクと対策

### 7.1 チャーン対策

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|---------|------|
| オンボーディング失敗による早期解約 | 高 | 中 | 導入後30日集中フォロー、オンボーディング完了基準の明確化 |
| 機能不足による競合乗り換え | 高 | 中 | 開発ロードマップとの連動、顧客要望の優先度管理 |
| BPO品質の低下 | 高 | 低 | BPOチームとのSLA設定、品質モニタリング体制 |
| 顧客担当者の異動・退職 | 中 | 高 | 複数キーパーソンとの関係構築、管理者権限の分散 |
| 利用率の低下（放置アカウント） | 中 | 高 | ヘルススコア監視、プロアクティブな利用促進 |

**重点対策：CS先行投資**

現在40数社の段階で問い合わせ増加・チャーン懸念が顕在化しているため、55期から56期にかけてCS人員を1名→2名に増員し、顧客数が100社を超える57期にはCS3名体制を確立する。受動的なサポート対応から、能動的なカスタマーサクセス活動への転換を55期中に実現する。

### 7.2 営業リソース不足への対応

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|---------|------|
| 人材採用の遅延 | 高 | 中 | 採用活動の前倒し、リファラル採用の強化、業務委託の活用 |
| 営業スキルの属人化 | 高 | 中 | セールスプレイブックの整備、ロールプレイング研修の定例化 |
| マネジメント層の不足 | 中 | 中 | 泉氏による組織基盤の構築、57期以降の外部採用 |
| パートナー管理の負荷増大 | 中 | 高 | パートナーポータルの整備、対応の定型化・自動化 |

**重点対策：営業プロセスの標準化**

55期〜56期の早い段階で、以下の営業資産を整備する。
- セールスプレイブック（業種別の提案ストーリー、FAQ、反論対応）
- ROI試算テンプレート（導入効果の定量化ツール）
- 提案書テンプレート（業種別・プラン別）
- デモシナリオ（5分版・15分版・30分版）

これにより、新規営業メンバーの立ち上がり期間を短縮し、営業品質の均一化を図る。

### 7.3 競合対策

#### 主要競合と差別化戦略

| 競合カテゴリ | 代表的サービス | ConPassの差別化ポイント |
|------------|-------------|---------------------|
| 電子契約サービス | クラウドサイン、GMOサイン | 紙契約も含めた一元管理、BPO統合 |
| 契約管理SaaS | LegalForce、Hubble | 中小企業向けの価格帯、BPO統合、IT導入補助金 |
| 文書管理システム | DocuWorks、Box | 契約書に特化した管理機能、AIアシスタント、3点照合 |

#### 競合への対抗施策

1. **BPO統合の深化**：純SaaS競合が模倣できないBPOサービスとの統合を継続的に強化
2. **バーティカル特化**：業界別パックにより、汎用SaaSでは対応できない業界固有ニーズに応える
3. **士業チャネルの独占**：士業向けの専用機能（3点照合、法令監視等）を先行して開発し、チャネルパートナーのスイッチングコストを高める
4. **価格競争への不参加**：BPO統合という付加価値を前面に出し、価格ではなく総合的なROIで勝負する

### 7.4 市場環境変化への適応

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|---------|------|
| 電子帳簿保存法等の法改正 | 中 | 高 | 法改正を営業機会に転換する体制整備、法改正対応ウェビナーの即時開催 |
| 景気後退によるIT投資抑制 | 高 | 中 | IT導入補助金の活用促進、ROI訴求の強化（年間360時間削減）、月額プランの柔軟化 |
| 大手SaaS企業の契約管理市場参入 | 高 | 低 | BPO統合・バーティカル特化の差別化を先行確立、士業チャネルの囲い込み |
| AI技術の急速な進化 | 中 | 高 | AIアシスタントの継続的強化、外部AI APIの活用による開発効率化 |
| パートナー企業の方針変更 | 中 | 低 | 特定パートナーへの依存度を制御、複数チャネルの並行育成 |

**重点対策：法改正を営業機会に変える仕組み**

電子帳簿保存法をはじめとする法改正は、契約管理ツールの導入機運を高める最大の追い風となる。法改正の動向を常時ウォッチし、以下のアクションを迅速に実行する体制を整える。

1. 法改正情報のキャッチアップ体制（士業パートナーからの情報入手）
2. 法改正対応機能の迅速な開発・リリース
3. 法改正対応ウェビナーの即日告知・1週間以内開催
4. 法改正対応ホワイトペーパーの公開
5. リスティング広告のキーワード即時追加
