### 1.2 ブレイクイーブンポイント分析

#### 固定費構造（年間）

| 項目 | 金額（千円/年） | 備考 |
|------|----------------|------|
| 業務委託費 | 10,800 | 90万円/月 x 12ヶ月 |
| インフラ費（GCP/Qdrant/OpenAI等） | 3,600 | 推定30万円/月 |
| 営業・マーケティング費 | 12,000 | 推定（後述コスト構造で詳細） |
| 管理費・その他 | 3,600 | 推定30万円/月 |
| **固定費合計** | **30,000** | |

#### ブレイクイーブン到達時期

```
BEP = 固定費 / 粗利率

前提:
- SaaS粗利率: 80%（インフラ費・サポート費控除後）
- 必要ARR = 30,000千円 / 0.80 = 37,500千円

55期ARR = 39,368千円 > 37,500千円
```

**結論: 55期（2026年）にブレイクイーブン到達見込み。**

ただし、開発投資（MS1〜MS3: 各3,000万円）を含めた場合:
```
固定費（開発投資込み） = 30,000 + 30,000 = 60,000千円/年
必要ARR = 60,000 / 0.80 = 75,000千円
→ 56期後半〜57期前半にBEP到達（56期ARR: 58,900千円、57期ARR: 79,604千円）
```

### 1.3 Unit Economics

#### CAC（顧客獲得コスト）

各施策のCACを加重平均で算出:

| 施策 | 受注獲得単価（CAC） | 年間獲得見込み社数（55期） | 構成比 |
|------|-------------------|------------------------|--------|
| 展示会 | 546,296円 | 18社 | 47% |
| ビジネスタンク | 500,000円 | 6社 | 16% |
| リスティング | 879,121円 | 2社 | 5% |
| 既存プレーン・ウェビナー等 | 200,000円（推定） | 12社 | 32% |

```
加重平均CAC = (546,296 x 18 + 500,000 x 6 + 879,121 x 2 + 200,000 x 12) / 38
           = (9,833,328 + 3,000,000 + 1,758,242 + 2,400,000) / 38
           = 16,991,570 / 38
           = 447,146円

→ 加重平均CAC ≒ 45万円
```

#### LTV（顧客生涯価値）

```
LTV = ARPA / チャーンレート

前提:
- ARPA = 500,000円/年
- 月次チャーンレート = 1.5%（SaaS業界中央値）
- 年間チャーンレート = 1 - (1 - 0.015)^12 = 16.6%

LTV = 500,000 / 0.166 = 3,012,048円 ≒ 301万円
```

#### LTV/CAC比率

```
LTV/CAC = 3,012,048 / 447,146 = 6.7倍
```

| 指標 | 値 | 目安 | 判定 |
|------|-----|------|------|
| CAC | 45万円 | - | - |
| LTV | 301万円 | - | - |
| LTV/CAC | 6.7倍 | 3倍以上 | 良好 |
| CAC回収期間 | 10.8ヶ月 | 12ヶ月以内 | 良好 |

```
CAC回収期間 = CAC / (ARPA / 12) = 447,146 / (500,000 / 12) = 447,146 / 41,667 = 10.7ヶ月
```

> ARPA100万円到達時のLTV/CAC: 6,024,096 / 447,146 = 13.5倍（非常に優秀）

### 2.4 ST比率向上による売上インパクト分析

ST比率を40%→75%に引き上げた場合の60期ARRインパクト:

```
【現状維持シナリオ（ST40%）】
318社 x 60% x 30,000 x 12 = 68,688千円（ライト）
318社 x 40% x 50,000 x 12 = 76,320千円（ST）
基本プラン合計 = 144,908千円

【ST75%シナリオ】
318社 x 25% x 30,000 x 12 = 28,620千円（ライト）
318社 x 75% x 50,000 x 12 = 143,100千円（ST）
基本プラン合計 = 171,720千円

差額 = 171,720 - 144,908 = +26,812千円/年（+18.5%）
```

> ST比率向上だけで年間約2,681万円のARR増加が見込める。

## 5. 収益性分析

### 5.1 粗利率の推移予測

粗利 = ARR - 直接費用（インフラ + BPOオペレーション）

| 期 | ARR（千円） | 直接費（千円） | 粗利（千円） | 粗利率 |
|----|-----------|-------------|-----------|--------|
| 55期 | 39,368 | 7,800 | 31,568 | 80.2% |
| 56期 | 58,900 | 11,340 | 47,560 | 80.7% |
| 57期 | 79,604 | 15,540 | 64,064 | 80.5% |
| 58期 | 101,609 | 19,680 | 81,929 | 80.6% |
| 59期 | 127,011 | 23,880 | 103,131 | 81.2% |
| 60期 | 158,764 | 28,020 | 130,744 | 82.3% |

```
直接費 = インフラ費 + BPOオペレーション費
55期: 3,600 + 4,200 = 7,800千円

55期: (39,368 - 7,800) / 39,368 = 80.2%
```

> SaaS事業として粗利率80%超を安定的に維持。スケールに伴い逓増傾向。

| 59期 | 103,131 | 110,035 | -6,904 | -5.4% |
| 60期 | 130,744 | 104,900 | +25,844 | +16.3% |

