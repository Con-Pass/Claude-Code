**修正方針:** 04_GTM・営業戦略のARPA推移は「ARPA引き上げ目標」（将来の目標シナリオ）であり、03_財務モデルのARPA（ベースシナリオ）とは計画の前提が異なる。最終版では「ベースシナリオARPA 50万円」と「ARPA引き上げ目標シナリオ」を明確に区分して記載する。

### 2.4 ST比率目標

**指定値:** MS0:10% → MS1:20% → MS2:35% → MS3:50% → MS4:60% → MS5:70%+

| セクション | MS0 | MS1 | MS2 | MS3 | MS4 | MS5 | 判定 |
|-----------|-----|-----|-----|-----|-----|-----|------|
| 02_プロダクト戦略 | 10% | 20% | 35% | 50% | 60% | 70%+ | **OK** |
| 03_財務モデル | - | - | - | - | - | - | 別形式で記載（期別ST比率） |
| 04_GTM・営業戦略 | - | - | - | - | - | - | 直接記載なし |

**不整合発見 [中程度]:**

- **03_財務モデル 2.3節のST比率:**
  55期40%→56期50%→57期60%→58期65%→59期70%→60期75%
  これはプロダクト戦略のMS別ST比率目標とは異なる体系（期別 vs MS別）で記載されているが、MSと期の対応関係を考慮すると概ね整合する。ただし、55期時点でST40%（財務モデル）vs MS0時点で10%（プロダクト戦略）に不一致がある。

**修正方針:** 03_財務モデルの55期ST40%は「55期の期間加重平均」であり、55期初頭（MS0）のST10%から55期末（MS1完了）のST20%への途中経過を期間平均したもの、という解釈は困難。03_財務モデルの売上構成は「ライト60%:ST40%」で計算されており、プロダクト戦略のMS0時点ST10%とは矛盾する。最終版では財務モデルのプラン構成比をプロダクト戦略のST比率と整合させる注記を付す。

### 2.5 マイルストーン時期

**指定値:** MS1:2026.05 → MS2:2026.09 → MS3:2027.03 → MS4:2027.09 → MS5:2028.03

| セクション | MS1 | MS2 | MS3 | MS4 | MS5 | 判定 |
|-----------|-----|-----|-----|-----|-----|------|
| 02_プロダクト戦略 | 2026.05 | 2026.09 | 2027.03 | 2027.09 | 2028.03 | **OK** |
| 03_財務モデル | 55期(2026) | 56期(2027) | 57期(2028) | 58期(2029) | 59期(2030) | **不整合あり** |

**不整合発見 [重要]:**

- **03_財務モデル 3.1節および4.1節のMS別開発投資計画で、MSと期の対応が異なっている。**
  - プロダクト戦略: MS1〜MS2は55期（2026年）内、MS3は55期末〜56期、MS4は56期〜57期、MS5は57期
  - 財務モデル: MS1=55期、MS2=56期、MS3=57期、MS4=58期、MS5=59期（1MS=1期で配置）

  プロダクト戦略ではMS1（〜2026.05）とMS2（〜2026.09）は同一期（55期）に含まれるが、財務モデルではMS1=55期、MS2=56期と別期に割り当てている。

**修正方針:** マイルストーンの期への配分はプロダクト戦略のタイムラインが正。ただし財務モデルでは「MS別の開発投資をどの期に計上するか」という会計処理の観点から期別に配分しており、開発投資の総額（各3,000万円 x 3 = 9,000万円）は一致している。最終版では「MSの完了時期」と「開発投資の期別配分」は別概念であることを明記する。

### 2.10 施策別ARR合計と顧客数ベースARRの乖離

**発見した重要な不整合:**

| 期 | 顧客数ベースARR（千円） | 施策別積上ARR（千円） | 乖離 |
|----|---------------------|---------------------|------|
| 55期 | 39,368 | 31,633 | +7,735 |
| 56期 | 58,900 | 77,861 | -18,961 |
| 57期 | 79,604 | 144,292 | -64,688 |
| 58期 | 101,609 | 213,489 | -111,880 |
| 59期 | 127,011 | 285,450 | -158,439 |

55期を除き、施策別積上ARRが顧客数ベースARRを大幅に上回っている。56期以降は施策別積上値が顧客数ベースの2〜2.2倍になっている。

**原因分析:** 施策別積上ARRは「各施策が理想的に実行された場合の最大ポテンシャル」であり、顧客数ベースARRは「現実的な顧客獲得とチャーンを考慮した保守的計画」と解釈される。03_財務モデルではこの二重構造が明示されている（1.1節で両数値を並記）。

**修正方針:** 最終版では「施策別積上ARR」は営業活動のKPI目標値、「顧客数ベースARR」は財務計画の基準値として、両者の位置づけと乖離の理由を明記する。

## 3. 発見した追加の不整合・矛盾

### 3.1 対象期間の不一致

| セクション | 記載の対象期間 |
|-----------|-------------|
| 01_市場調査 | 54期（2025年）〜 59期（2030年） |
| 02_プロダクト戦略 | 明示なし（MS0〜MS5: 2026.02〜2028.03） |
| 03_財務モデル | 55期（2026年）〜 60期（2031年） |
| 04_GTM・営業戦略 | 55期〜59期（暗黙） |

**修正方針:** 統一的に「55期（2026年）〜60期（2031年）」を5ヵ年計画の対象期間とし、54期以前は「前提・現状」として扱う。プロダクト戦略のMS5（2028.03）以降の59期・60期については、プラットフォーム安定期として位置づける。

### 3.2 GTM・営業戦略のARR目標表記

- 04_GTM・営業戦略 1.1節で「ARR 285,450千円（59期末）」と記載しているが、これは施策別積上ARRの59期合計値であり、財務モデルの59期ARR 127,011千円とは異なる。
- 最終版では「施策別目標ARR」と「財務計画ARR」を明確に区分する。

### 3.3 チャーンレートの前提不一致

| セクション | チャーンレート前提 |
|-----------|----------------|
| 03_財務モデル（売上計画） | 年間6〜8%（月次0.5〜0.7%） |
| 03_財務モデル（Unit Economics） | 月次1.5%（年間16.6%）※LTV計算用 |
| 04_GTM・営業戦略 | 月次2.0%→1.2%（段階的改善） |

**修正方針:** 財務モデルの売上計画で使用する月次チャーンレート（0.5〜0.7%）を基準とし、LTV計算用の1.5%は「業界中央値（保守的参考値）」として区分する。GTM・営業戦略の月次2.0%は「CS体制構築前の現状値」として位置づけ、改善後の目標値が財務モデルの前提と整合するよう記載を修正する。
