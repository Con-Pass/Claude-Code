## 1. ログインとログアウト

### ログイン

1. ブラウザで ConPass の URL を開きます。
2. **メールアドレス**と**パスワード**を入力し、「ログイン」ボタンを押します。
3. 組織によっては **Google / Microsoft アカウント**でのソーシャルログイン、または **SSO（シングルサインオン）** が使用できます。管理者に確認してください。

> **ヒント**: パスワードを忘れた場合はログイン画面の「パスワードを忘れた方」リンクからリセット手続きを行ってください。

### ログアウト

1. 画面右上のユーザー名または設定メニューから「ログアウト」を選択します。
2. 機密情報を扱う端末では、作業後に必ずログアウトしてください。

## 2. ダッシュボード

ログイン後に表示されるトップ画面です。

| 表示エリア | 内容 |
|----------|------|
| **契約書総数** | アクセス可能なディレクトリにある契約書の合計件数 |
| **有効契約書数** | 現在有効な契約書の件数 |
| **期限が近い契約書** | 更新期限が迫っている契約書の件数と一覧 |
| **最近の契約書** | 最近アップロードまたは更新された契約書の一覧 |

期限が近い契約書や最近の契約書の行をクリックすると、その契約書の詳細画面に移動します。

## 3. 契約書の取り込み（アップロード）

### 対応ファイル形式

- PDF（推奨）
- Word（`.docx` / `.doc`）
- 画像ファイル（`.jpg` / `.png` など）

### アップロード手順

1. 画面上部のナビゲーションから **「契約書一覧」** を開きます。
2. 画面右上の **「ファイルを取り込む」** ボタンをクリックします。
3. ダイアログが開いたら、**保存先ディレクトリ**を選択します。
4. ファイルをドラッグ＆ドロップ、または「ファイルを選択」でファイルを指定します。
5. 「取り込む」ボタンを押します。

### アップロード後の自動処理

ファイルをアップロードすると、バックグラウンドで以下の処理が自動実行されます。

```
ファイル保存
  ↓
テキスト抽出（OCR）
  ↓
メタ情報の自動抽出（会社名・契約日・終了日など）
  ↓
AIによる全文インデックス作成（セマンティック検索向け）
```

抽出処理には通常 **30秒〜2分**かかります。処理の進捗は契約書詳細画面の上部に表示されます。

> **注意**: 処理完了前にメタ情報タブを開くと、まだ値が入っていない場合があります。処理完了後に再確認してください。

## 4. 契約書一覧の操作

### 一覧画面でできること

| 操作 | 方法 |
|-----|------|
| **キーワード検索** | 検索ボックスにキーワードを入力してEnter |
| **ディレクトリで絞り込み** | 左サイドバーのフォルダ（ディレクトリ）をクリック |
| **列の並べ替え** | 列ヘッダーをクリック（昇順/降順） |
| **ページ切り替え** | 一覧下部のページネーションを使用 |
| **詳細画面へ移動** | 行をクリック |

### ステータスについて

契約書には以下のステータスがあります。

| ステータス | 説明 |
|----------|------|
| **有効** | 現在有効な契約書 |
| **期限切れ** | 契約期間が終了した契約書 |
| **キャンセル** | 解約された契約書 |
| **ゴミ箱** | 削除予定の契約書（完全削除前） |

## 5. 契約書の詳細確認

契約書の行をクリックすると詳細画面が開きます。詳細画面には **3つのタブ**があります。

### タブ①：ファイルプレビュー

アップロードしたファイルをブラウザ内でプレビューできます。

- PDF はページをめくりながら確認できます。
- ファイルが複数ある場合（例：本契約 + 覚書）は、タブ内のファイル切り替えメニューで選択できます。
- 右上の「ダウンロード」ボタンから元ファイルをダウンロードできます。

### タブ②：テキスト全文

OCRで抽出された契約書本文のプレーンテキストが表示されます。

- **Ctrl+F**（Mac は Cmd+F）でページ内テキスト検索ができます。
- 抽出精度が低い場合は「🔄 再抽出」ボタンを押すと再度 AI 抽出が実行されます。

### タブ③：メタ情報

AIが自動抽出したメタ情報（会社名・契約日・終了日など）が表示されます。

詳細は [6. メタ情報の確認と編集](#6-メタ情報の確認と編集) を参照してください。

### 再抽出ボタン

タブバー右端にある **「🔄 再抽出」** ボタンを押すと、テキスト抽出とメタ情報抽出を再実行します。

- ボタンを押すと「抽出中...」と表示され、完了後に各タブの内容が自動更新されます。
- ファイル内容を更新した後や、初回抽出の精度が低かった場合にご利用ください。

## 6. メタ情報の確認と編集

### 自動抽出されるメタ情報（例）

| 項目 | 説明 |
|-----|------|
| 契約書名 | 契約書のタイトル |
| 会社名（甲・乙） | 契約当事者の会社名 |
| 契約日 | 契約締結日 |
| 契約開始日 | 契約の有効開始日 |
| 契約終了日 | 契約の有効終了日 |
| 契約種別 | 業務委託・秘密保持・売買など（AI自動分類） |
| 自動更新 | 自動更新条項の有無 |
| 裁判所 | 管轄裁判所 |

### 契約種別（3階層分類）

ConPass は PDF をアップロードすると、AIが契約書を以下の3階層で自動分類します。

| 大分類（contract_category） | 中分類（contract_type）の例 |
|---------------------------|--------------------------|
| 取引契約 | 基本契約書、業務委託契約書、売買契約書 |
| 知的財産・情報契約 | 秘密保持契約書、ライセンス契約書 |
| 人事・労務契約 | 雇用契約書、派遣契約書 |
| 不動産・賃貸借契約 | 賃貸借契約書、リース契約書 |
| 紛争解決・合意契約 | 和解契約書、合意書・覚書 |

### メタ情報の編集手順

1. 「メタ情報」タブを開きます。
2. 修正したい項目の値をクリックします。
3. テキストを修正し、**Enterキー**または**チェックマーク**で保存します。

> **AI抽出値の確認**: AI が抽出した値には信頼度スコアが付いています。値が正しければ「確認済み」マークを付けることで、品質管理に役立ちます。

## 7. コメントの投稿

契約書の詳細画面でチームメンバーとのコミュニケーションができます。

### コメントの追加

1. 詳細画面下部の**コメントエリア**を開きます。
2. テキストボックスにコメントを入力します。
3. 「送信」ボタンを押します。

### 活用例

- 「2024年3月更新時に条件交渉の余地あり」
- 「法務確認待ち：第5条の損害賠償条項について」
- 「取引先担当者変更 → 田中さんから鈴木さんへ」

## 8. ワークフロー（承認フロー）

ワークフローを使うと、契約書の確認・承認を複数人で順番に行えます。

### ワークフローの開始

1. 契約書の詳細画面を開きます。
2. 「ワークフロー」メニューから「**ワークフローを開始**」を選択します。
3. 使用するワークフローテンプレートを選択します。
4. 各ステップの担当者を確認（または変更）して「開始」を押します。

### 自分にタスクが割り当てられた場合

1. ダッシュボードまたは通知から「承認依頼」を確認します。
2. 対象の契約書詳細画面を開きます。
3. 内容を確認後、「**タスク完了**」ボタンを押します。
   - 必要に応じてコメントを追加できます。
   - 内容に問題がある場合は「**差し戻し**」ボタンで前のステップに戻せます。

### ワークフローのステータス確認

契約書詳細画面の「ワークフロー」タブで、現在のステップと各担当者の完了状況が確認できます。

## 9. AIアシスタントで検索・分析

画面右下のチャットウィジェット（💬アイコン）をクリックすると、AI アシスタントに話しかけられます。

### できること

**契約書の検索**
```
「2025年に期限が切れる業務委託契約書を探して」
「A社との秘密保持契約の内容を教えて」
「自動更新条項がある契約書はどれ？」
```

**複数契約書の比較**
```
「#30の基本契約と#36の覚書の違いを教えて」
「GlobalData社との契約をすべてリスト化して」
```

**メタ情報の確認・更新**
```
「#25の契約終了日を確認して」
「Reuters社との契約書の契約種別を更新して」
```

**データのエクスポート**
```
「2024年中に期限切れになる契約書のCSVを作成して」
「全ライセンス契約書の会社名と期限の一覧を出して」
```

### 使い方のコツ

- **具体的に指示する**：「業務委託契約」「秘密保持」など契約種別を入れると精度が上がります。
- **番号で指定**：「#30の契約」のように ID 番号で直接指定できます。
- **関連を尋ねる**：「#30に関連する契約書は？」と聞くと、親子関係にある覚書なども表示されます。

### AIが答えられないこと

- ConPass に登録されていない外部情報
- 法律的な判断（弁護士への相談を推奨）
- ファイルの修正や削除（直接画面から操作してください）

## 10. 契約書のエクスポート

### 個別エクスポート

契約書詳細画面の右上にある「エクスポート」メニューから選択できます。

| 形式 | 内容 |
|-----|------|
| **PDF** | 契約書のPDFファイルをダウンロード |
| **Word** | Word形式でダウンロード |
| **ZIP** | 関連ファイルをまとめてダウンロード |

### 一括エクスポート（CSV）

1. 契約書一覧画面で「エクスポート」ボタンを押します。
2. 「メタデータCSVダウンロード」を選択します。
3. 一覧に表示中の契約書のメタ情報がCSVでダウンロードされます。

AI アシスタントに「CSVを作成して」と依頼することでも、条件付きのCSVが作成できます。

## 11. 通知の確認

### 通知の種類

| 種類 | タイミング |
|-----|---------|
| **ワークフロー通知** | 承認タスクが自分に割り当てられたとき |
| **契約期限通知** | 設定した日数前に期限が迫った契約書がある場合 |

### 通知の確認方法

- **アプリ内通知**：画面右上のベルアイコン（🔔）をクリック
- **メール通知**：登録メールアドレスに届いたメールを確認

> 通知の受信設定（アプリ内のみ / メールのみ / 両方）は、設定画面で変更できます。詳細は管理者に確認してください。

## 12. よくある質問

**Q: ファイルをアップロードしたがメタ情報が空のまま**
→ 処理に時間がかかっています。1〜2分待ってからページを再読み込みしてください。それでも空の場合は「🔄 再抽出」ボタンを試してください。

**Q: OCRのテキストに誤りがある**
→ 手書き文字や特殊なフォントはOCR精度が下がることがあります。「🔄 再抽出」を試すか、テキスト全文タブで直接修正してください。

**Q: AIアシスタントが「見つかりませんでした」と返答する**
→ 対象の契約書が自分のアクセス可能なディレクトリにない可能性があります。管理者に権限設定を確認してもらってください。

**Q: 誤って削除してしまった**
→ 削除した契約書は「ゴミ箱」に移動します。完全削除前であれば復元できます。管理者にご連絡ください。

**Q: ワークフローでコメントを付けずに承認してしまった**
→ ワークフロー履歴にコメント追加は承認後もできます。詳細画面のワークフロータブからコメントを追加してください。
